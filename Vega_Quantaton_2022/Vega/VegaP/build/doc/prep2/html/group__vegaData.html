Computes cost-of-carry rate curve \(c=(c(t))_{t\geq t_0}\) for the Black model. In this model, the log of spot price evolves as

\[ \log S_t = \log S(t_0) + X_t, \quad t\geq t_0, \]

where \(X=(X_t)\) is an OU (Ornstein-Uhlenbeck) process driven by Brownian motion \(B=(B_t)\):

\[ dX_t = (\theta - \lambda X_t) dt + \sigma dB_t, \quad X(t_0) = 0. \]

We recall that the forward price curve has the form:

\[ F(t) = S(t_0) \exp(c(t)(t-t_0)) = \mathbb{E}(S(t)), \quad t\geq t_0. \]

The computations show that

\[ q(t) = \theta \frac{1-e^{-\lambda(t-t_0)}}{\lambda(t-t_0)} + \frac{\sigma^2}2 \frac{1-e^{-2\lambda(t-t_0)}}{2\lambda(t-t_0)}, \quad t\geq t_0. \]

Parameters
dTheta\(\theta\) The drift term.
dLambda\(\lambda\geq 0\) The mean reversion level.
dSigma\(\sigma\geq 0\) The volatility.
dInitialTime\(t_0\) The initial time.
Returns
The cost-of-carry rate curve in Black model.

std::function<double(double)> vega::discountVasicek ( double  dTheta,
double  dLambda,

double  dR0,
double  dInitialTime

We recall that the discount curve has the form:

\[ D(t) = e^{-\gamma(t)(t-t_0)}, \quad t\geq t_0. \]

In the Vasicek model, yield curve \(\gamma = \gamma(t)\) is given by

\[ \gamma(t) = r(t_0) A(t) + \frac{\theta}{\lambda}(1-A(t)) - \frac{\sigma^2}{2\lambda^2} (1-2A(t)+B(t)), \quad t\geq t_0, \]

where

\[ A(t) = \frac{1-e^{-\lambda(t-t_0)}}{\lambda(t-t_0)}, \quad B(t) = \frac{1-e^{-2\lambda(t-t_0)}}{2\lambda(t-t_0)}. \]

Parameters
dTheta\(\theta\) The drift.
dLambda\(\lambda\geq 0\) The mean-reversion rate.
dSigma\(\sigma\geq 0\) The volatility.
dR0\(r(t_0)\) The initial short-term interest rate.
dInitialTime\(t_0\) The initial time.
Returns
The discount curve for the Vasicek model of interest rates.

Computes the curve of forward prices ("clean" or "dirty") for an annuity. The annuity pays coupons \(q \delta t\) at times \((t_i)_{i=1,\dots,M}\) such that

\[ t_0 < t_1 \leq t_0+\delta t, \quad t_{i+1}-t_i= \delta t, \quad t_M = T. \]

The buyer pays forward price \(F(t)\) at delivery time \(t\) and then receives coupons \(q\delta t\) at payments times \(t_i>t\).

Parameters
dRate\(q\) The annuity rate.
dPeriod\(\delta t\) The time interval between payments.
dMaturity\(T\) The maturity.
rDiscountThe discount curve.
bCleanThis parameter controls the output. If its value is true, then we compute "clean" prices. Otherwise, we compute "dirty" prices that include the accrued interest from the next coupon.
Returns
The forward prices ("clean" or "dirty") for an annuity.

Returns the stationary implied volatility curve for the Hull and White (Vasicek) model of interest rates:

\[ \Sigma(s,t) = \sigma \frac{1-\exp(-\lambda(t-s))}{\lambda} \sqrt{\frac{1-\exp(-2\lambda(s-t_0))}{2\lambda(s-t_0)}}, \quad t_0\leq s\leq t, \]

where \(s\) is the maturity of the option written on the zero-coupon bond with maturity \(t\).

Parameters
dSigma\(\sigma\geq 0\) The short-term volatility.
dLambda\(\lambda\geq 0\) The mean-reversion rate.
dInitialTime\(t_0\) The initial time.
Returns
The stationary implied volatility curve in the Hull-White model as the function of option maturity (first arg) and bond maturity (second arg).

Computes yield curve \(\gamma=(\gamma(t))_{t\geq t_0}\) for the Vasicek model, where short-term interest rate \(r=(r_t)\) is an OU (Ornstein-Uhlenbeck) process driven by Brownian motion \(B=(B_t)\):

\[ dr_t= (\theta - \lambda r_t) dt + \sigma dB_t, \quad t\geq t_0. \]

We recall that the discount curve has the form:

\[ D(t) = e^{-\gamma(t)(t-t_0)} = \mathbb{E}(e^{-\int_{t_0}^t r_s ds}), \quad t\geq t_0. \]

Computations show (please, check) that

\[ \gamma(t) = r(t_0) A(t) + \frac{\theta}{\lambda}(1-A(t)) - \frac{\sigma^2}{2\lambda^2} (1-2A(t)+B(t)), \quad t\geq t_0. \]

where

\[ A(t) = \frac{1-e^{-\lambda(t-t_0)}}{\lambda(t-t_0)}, \quad B(t) = \frac{1-e^{-2\lambda(t-t_0)}}{2\lambda(t-t_0)}. \]

Parameters
dTheta\(\theta\) The drift.
dLambda\(\lambda\geq 0\) The mean-reversion rate.
dSigma\(\sigma\geq 0\) The volatility.
dR0\(r(t_0)\) The initial short-term interest rate.
dInitialTime\(t_0\) The initial time.
Returns
The yield curve for the Vasicek model of interest rates.
