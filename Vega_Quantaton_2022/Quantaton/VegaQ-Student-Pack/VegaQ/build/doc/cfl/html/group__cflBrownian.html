Detailed Description

This module contains implementation of the basic financial model where the state process is a one-dimensional Brownian motion.

typedef std::function<Model(const std::vector<double> &rVar, const std::vector<double> &rEventTimes, double dInterval)> cfl::TBrownian

Returns cfl::Model representing a Brownain motion given

  • rVar The vector of variances of the Brownian motion. rVar[i] defines the average variance between rEventTimes[0] and rEventTimes[i]. This vector has the same size as the vector of event times.
  • rEventTimes The vector of event times in the model.
  • dInterval The width of the interval of initial values for the Brownian motion.
