typedef std::function<void(Slice &rSlice, unsigned iEventTime)> cfl::TRollback

Implements the rollback operator: replaces rSlice with its price at iEventTime.

  • rSlice Before the function call it defines the input payoff. After the function call it defines the price of the input payoff at the event time with index iEventTime.
  • iEventTime The index of the target event time for the rollback.
