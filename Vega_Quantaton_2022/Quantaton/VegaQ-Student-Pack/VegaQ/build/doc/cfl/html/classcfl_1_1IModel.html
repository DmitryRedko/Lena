Detailed Description

Interface class for financial models.

Interface class IModel defines the model-specific behavior of a Slice object.

Transforms rSlice into the equivalent Slice object that also depends on the state processes with indexes rStates. This function is used to define arithmetic operations between Slices dependent on different state processes.

Parameters
rSliceThe representation of some random variable in the model. After this operation, the random variable will also be dependent on the state processes with indexes rStates.
rStatesAdditional indexes of state processes.
