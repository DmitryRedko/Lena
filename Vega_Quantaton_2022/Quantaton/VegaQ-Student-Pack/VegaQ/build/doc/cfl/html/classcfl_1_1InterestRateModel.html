◆ assignEventTimes()

void cfl::InterestRateModel::assignEventTimes ( const std::vector< double > &  rEventTimes)

Resets the vector of event times to rEventTimes and removes all path dependent state processes. After this operation, the member function model() will return a reference to a different "standard" implementation of IModel.

Parameters
rEventTimesThe new vector of event times for the model. The front element of this vector should equal the initial time of the model.
